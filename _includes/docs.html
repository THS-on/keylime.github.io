I want to try Keylime!

Grab our all-in-one Ansible role with a TPM emulator


git clone https://github.com/keylime/ansible-keylime-tpm-emulator/

Use our Vagrantfile to stand up a VM


$ vagrant up --provision
$ vagrant ssh
$ sudo -s
